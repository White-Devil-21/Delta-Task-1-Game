PIANO TILE GAME

This is a memory based game

The rules are very simple. A random tile will light up and you have to click it. If you click them successfully, then along with the previous round tiles another tile will light up. You can click the tiles in thev same order. If you failed to click all the tiles, then you will lose the game For each tile the maximum clicking time is 2 sec. If you take more time than that, you may lose. You can select either 4x4 level or 6x6 level
